# FinanceApp Frontend

This is the frontend for the FinanceApp, built with Angular 17.

## Architectural Patterns
- **Standalone Components:** All components are standalone.
- **Signals:** Reactive state management using Angular Signals.
- **Computed:** Derived state using `computed()`.
- **Services:** Logic and data fetching encapsulated in `FinanceService`.

## Coding Conventions
- Use English for all code symbols (variables, methods, signals).
- Follow Angular style guide.
- Use Tailwind CSS for styling.
- Use Phosphor Icons for iconography.
