# FinanceApp Backend

This is the backend for the FinanceApp, built with FastAPI.

## Architectural Patterns
- **API Endpoints:** Located in `app/api/endpoints/`.
- **Schemas:** Pydantic models in `app/schemas/schemas.py`.
- **Database:** Simple in-memory simulation in `app/db/session.py`.

## Coding Conventions
- Use English for all code symbols (variables, functions, classes).
- Follow PEP 8 for Python code styling.
- Use Pydantic for data validation and serialization.
